# Photino.Fun.MudBlazor

## Description
This is a F# project template for the create [Photino.net](https://www.tryphotino.io/) desktop application with using [Fun.Blazor](https://slaveoftime.github.io/Fun.Blazor.Docs/) and [MudBlazor](https://mudblazor.com/).

### Prerequisites
.Net SDK 10.0

### Installation

Clone the repo: https://github.com/poimenov/Photino.Fun.MudBlazor.git

Navigate to the Photino.Fun.MudBlazor folder

Remove the .git folder

Run dotnet new install .\

```bash
cd Photino.Fun.MudBlazor
rm -rf .git
dotnet new install .\
```
If you're using a Linux or macOS operating system, use a forward slash: 
```bash
dotnet new install ./
```

### Usage
```bash
dotnet new photino-fun-mudblazor -o MyApp
cd MyApp
dotnet run
```
